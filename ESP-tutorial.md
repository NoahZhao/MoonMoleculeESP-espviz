## 0 前言

静电势（Electrostatic Potential, ESP）与电子密度一样，是重要的三维实空间函数，在计算化学领域应用广泛，涉及分子反应位点、弱相互作用、凝聚相物理性质等。本文旨在系统梳理静电势的各种计算与分析方法，帮助读者建立相关知识框架。

文中大量引用作者之前撰写的博文，并提及 **Multiwfn** 程序（官网：http://sobereva.com/multiwfn）可实现文中所列所有分析方法。Multiwfn 支持与主流量子化学程序联用，是静电势分析领域最强大且易用的工具之一。

## 2 分子表面静电势的定量分析

Multiwfn 的主功能 12 支持对分子表面静电势进行多种定量分析，算法详见：J. Mol. Graph. Model., 38, 314 (2012)。

### 2.1 表面静电势相关的描述符

- GIPF 描述符：最大值、最小值、平均值、方差 σ²、电荷平衡度 ν、νσ² 等  
- **分子极性指数（MPI）**：作者提出的衡量分子极性的指标，适用于偶极矩为零的体系（如 18 碳环）。  
- **极性/非极性表面积**：以 |ESP| > 10 kcal/mol 为界划分。

相关博文：《使用Multiwfn预测晶体密度、蒸发焓、沸点、溶解自由能等性质》（http://sobereva.com/337）、《谈谈如何衡量分子的极性》（http://sobereva.com/518）

### 2.2 表面静电势的极值点分析

可搜索全局/局部极大点和极小点，用于预测反应位点、分子间相互作用位置和强度，甚至预测共晶形成（Chem. Sci., 2, 883 (2011)）。

操作示例：《使用Multiwfn的定量分子表面分析功能预测反应位点、分析分子间相互作用》（http://sobereva.com/159）

### 2.3 面积分布统计

作者提出表面静电势分布面积统计方法（J. Phys. Org. Chem., 26, 473 (2013)；Struct. Chem., 25, 1521 (2014)），可定量对比不同分子的静电势特征。

操作示例：《使用Multiwfn结合VMD分析和绘制分子表面静电势分布》（http://sobereva.com/196）

### 2.4 局部原子表面上的静电势分析

将分子表面划分为各原子对应的局部表面，分别计算静电势描述符。示例：《谈谈怎么计算“原子的静电势”》（http://sobereva.com/641）

### 2.5 特征区域的静电势分析

针对 σ-穴、π-穴、孤对电子区域等，可计算其面积和平均静电势，并可视化。操作见 Multiwfn 手册 4.12.10 节。

### 2.6 表面静电势的盆分析

借鉴 AIM 理论，将分子表面划分为不同盆，分析各盆的静电势特征。见 Multiwfn 手册 4.12.11 节。

## 0 前言

“原子的静电势”是一个含糊不清的说法。静电势是三维空间的函数，不同坐标处数值不同，并不针对原子本身。本文介绍三种有实际意义的、基于静电势计算的与原子相关的量：

1. **原子的拟合静电势电荷**  
2. **原子局部范德华表面上的静电势平均值**  
3. **原子核位置（扣除自身贡献）的静电势**

读者可根据自己的研究目的选择相应方法。文中示例使用 **Multiwfn** 程序（免费下载：http://sobereva.com/multiwfn），建议先阅读《Multiwfn FAQ》（http://sobereva.com/452）和《Multiwfn入门tips》（http://sobereva.com/167）了解基本操作。

## 1 原子的拟合静电势电荷

拟合静电势电荷是一类原子电荷计算方法，使得基于原子电荷粗略计算出的分子范德华表面附近区域的静电势，尽可能接近基于电子密度严格计算得到的静电势。

- 常见方法：Merz-Kollman、CHELPG、RESP 等  
- 工具：Multiwfn 可基于量子化学程序（如 Gaussian）输出的波函数文件计算  
- 详细教程：《RESP拟合静电势电荷的原理以及在Multiwfn中的计算》（http://sobereva.com/441）

**原理简述**：某原子的拟合静电势电荷数值，取决于该原子周围范德华表面附近的拟合点上的静电势。这些点上静电势整体越正，该原子的拟合电荷通常越正，反之亦然。

> 本文不再重复举例，请参考前述博文。

## 2 原子局部范德华表面的静电势平均值

分子的范德华表面（通常定义为电子密度=0.001 a.u.的等值面）上的静电势分布可通过作图或统计量进行分析。

**局部分子表面分析**：将分子范德华表面划分为各个原子（或原子片段）所属的局部区域，计算每个局部区域上的静电势统计量（平均值、方差、最大/最小值等），从而获得与原子相关的信息。

### 示例：丙烯醛

**体系**：丙烯醛（三个碳原子：羰基碳、α-碳、β-碳）  
**已知**：羰基碳最容易受到硬亲核试剂进攻。

#### 操作步骤

1. 使用 Gaussian 在 B3LYP/6-31G** 级别计算得到波函数文件 `acrolein.wfn`（已包含在 Multiwfn 示例目录 `examples\acrolein.wfn`）。
2. 启动 Multiwfn，输入：
   ```
   examples\acrolein.wfn
   12   // 定量分子表面分析
   0    // 开始分析
   ```
3. 在后处理菜单中选择：
   ```
   11   // Output surface properties of each atom
   ```

#### 部分输出结果解读

- **原子 2（羰基碳）**：局部表面总面积 4.33 Å²，其中正静电势面积 3.67 Å²，负静电势面积 0.65 Å²。静电势平均值 **4.656 kcal/mol**。
- **原子 3（α-碳）**：静电势平均值 **1.309 kcal/mol**。
- **原子 6（β-碳）**：静电势平均值 **2.215 kcal/mol**。

羰基碳的局部表面静电势平均值最高，与其最易吸引硬亲核试剂的性质一致。

> 注意：某些量显示为 `NaN`，表示不存在对应区域（如某原子局部表面全为正静电势，则负区平均值为 NaN）。

#### 可视化

选择选项 11 后输入 `y`，Multiwfn 会生成 `locsurf.pdb` 文件。将该文件载入 VMD，根据 B 因子（对应原子序号）着色，可直观查看各原子局部表面划分是否合理。

## 3 原子核位置的静电势

在原子核处直接计算静电势会得到无穷大，因为自身原子核的贡献发散。实际有意义的是 **扣除该原子核自身贡献后的静电势**，即该原子核与体系其它部分的静电相互作用能。

### 意义

- 对于 **氢原子**：氢核处的静电势与质子解离难易程度相关。数值越负，质子被吸引越强，越难解离，pKa 可能越大。
- 注意：该值仅反映静电部分，实际 pKa 还涉及电子结构弛豫、溶剂效应等，因此只存在松散的相关性。

### 示例：甲酸、乙酸、乙醇

**波函数文件**（B3LYP/6-311G** 级别优化）：  
下载地址：http://sobereva.com/attach/641/file.rar

#### 甲酸（formic_acid.fch）

- 计算羧基氢（原子2）：
  ```
  Multiwfn formic_acid.fch
  1    // 计算一个点的属性
  a2   // 原子2的核位置
  ```
  输出：
  ```
  Total ESP without contribution from nuclear charge of atom 2:
  -0.9478 a.u.  ( -594.7 kcal/mol)
  ```
- 计算与碳相连的氢（原子5）：
  ```
  a5
  ```
  输出：**-1.0628 a.u.**

→ 原子5的核处静电势更负，更难解离，与常识一致。

#### 乙酸（acetic_acid.fch）

- 计算羧基氢：结果为 **-0.9589 a.u.**  
- 比甲酸的 -0.9478 更负，对应乙酸 pKa（4.756）大于甲酸（3.745）。

#### 乙醇（ethanol.fch）

- 计算羟基氢：结果为 **-1.0153 a.u.**  
- 比乙酸的羧基氢负得多，对应乙醇 pKa（15.9）远大于乙酸。

## 总结与对比

| 方法 | 含义 | 典型应用 | 与化学性质的关系 |
|------|------|----------|------------------|
| 拟合静电势电荷 | 原子核位置的点电荷，拟合范德华表面附近静电势 | 分子力场、电荷分布分析 | 反映原子对分子外部静电势的贡献 |
| 原子局部表面静电势平均值 | 原子所属范德华表面区域的静电势统计量 | 预测反应位点、原子间比较 | 与亲电/亲核反应倾向相关 |
| 原子核处静电势（扣除自身） | 原子核与其余部分的静电作用能 | 估计 pKa 相对大小、质子解离趋势 | 数值越负，质子越难解离 |

> 拟合静电势电荷与原子局部表面静电势平均值在原理上存在一定正相关；而原子核处静电势与前两者没有直接联系，除非原子所处化学环境相似。
