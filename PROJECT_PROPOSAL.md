# MoonBit 分子静电势可视化工具项目书

## 1. 项目名称

基于 MoonBit 的分子静电势可视化与验证工具

英文名称：MoonBit Molecular Electrostatic Potential Visualization Toolkit

## 2. 项目简介

本项目拟使用 MoonBit 语言实现一个轻量级分子静电势可视化工具，用于计算、展示和验证分子周围空间及分子表面上的静电势分布。项目面向计算化学教学、小分子电荷模型检查、静电势拟合结果展示，以及量子化学后处理数据的可视化分析。

工具当前包含两类计算模型：

1. 基于原子部分电荷的点电荷近似模型：
   \[
   \Phi(\mathbf r)=k\sum_i \frac{q_i}{|\mathbf r-\mathbf R_i|}
   \]
   该模型适合快速展示力场电荷、RESP/ESP 拟合电荷或教学示例，但不能等同于严格量子化学分子静电势。

2. 基于核电荷与电子密度网格的分子静电势后处理模型：
   \[
   V(\mathbf r)=k\left[\sum_A \frac{Z_A}{|\mathbf r-\mathbf R_A|}
   -\int \frac{\rho(\mathbf r')}{|\mathbf r-\mathbf r'|}d\mathbf r'\right]
   \]
   该模型符合量子化学中分子静电势的公认定义。项目本身不求解 Hartree-Fock 或 DFT 波函数，但可接收外部量子化学程序生成的电子密度网格，进行静电势积分与可视化。

## 3. 项目方向与适用场景

项目方向：科学计算可视化、计算化学后处理、MoonBit 数值计算应用、分子性质可视化。

适用场景包括：

- 计算化学与分子模拟教学中展示静电势正负区域、极性分布和电荷模型差异。
- 对小分子力场电荷、RESP/ESP 拟合电荷、Mulliken/NPA 等原子电荷模型进行快速检查。
- 对外部量子化学程序输出的电子密度网格进行 MESP 后处理与切片可视化。
- 在分子表面上分析静电势分布，为反应位点、氢键/卤键相互作用、分子识别等问题提供直观参考。
- 作为 MoonBit 在科学计算与可视化方向的示范项目。

## 4. 拟实现的核心功能

1. 分子输入与基础数据结构
   - 支持原子元素、三维坐标、部分电荷输入。
   - 支持核电荷 `Z_A` 与电子密度网格输入。
   - 支持常见小分子示例，如水、氨、甲烷、二氧化碳、甲醛等。

2. 静电势计算
   - 实现原子点电荷库仑势计算，用于快速预览与电荷模型教学。
   - 实现严格 MESP 后处理公式中的核电荷项与电子密度积分项。
   - 支持在二维平面网格上计算静电势分布。
   - 对采样点过近导致的库仑奇异性进行显式报错或排除，而不是隐藏或任意截断。

3. 分子表面定义
   - 对严格 MESP 表面，采用量子化学常用电子密度等值面定义，默认参考 `rho = 0.001 a.u.`，并保留 `0.002 a.u.` 作为常见替代值。
   - 对快速交互式演示，可使用范德华球并集表面作为近似展示，但在界面和文档中明确其不是严格电子密度等值面。

4. 可视化输出
   - 输出 CSV、PPM、SVG 等轻量格式。
   - 提供交互式 3D 前端，显示分子、切片平面、范德华轮廓和颜色图例。
   - 颜色映射以 0 为中心，负电势显示为红色，正电势显示为蓝色，中性附近显示为白色。
   - 支持固定对称色标，避免不同分子或不同表面因自动缩放而产生误导性颜色对比。

5. 科学说明与验证
   - 在文档中明确区分“严格量子化学 MESP”和“点电荷近似 ESP”。
   - 提供单元测试覆盖核电荷项、电子密度积分符号、电子数积分、单位换算和颜色映射。
   - 提供与外部量子化学软件或成熟工具对比的验证流程。

## 5. 项目原创性说明

本项目为原创项目。

项目使用 MoonBit 语言重新设计和实现分子静电势计算核心、数据结构、可视化导出和交互式示例页面。项目没有直接移植已有开源项目源码，也不复制第三方实现。

项目参考的是公开的量子化学理论公式和分子静电势分析方法，包括分子静电势的标准定义、电子密度等值面分子表面定义、ESP/RESP 电荷拟合思想，以及常见可视化颜色约定。相关理论参考包括项目中的 `ESP-tutorial.md`。

## 6. 是否为移植项目及参考项目说明

本项目不是移植项目，因此无需声明被移植项目。

项目开发过程中曾参考若干开源项目和工具的功能边界与问题设定，但未复制其代码：

- multipoles：参考多极展开和点电荷示例思想，许可证为 MIT。
- resp：参考 RESP/ESP 电荷拟合背景，许可证为 BSD-3-Clause。
- esp-surface-generator：参考静电势作为分子机器学习目标的应用场景，许可证为 Apache-2.0。

上述项目仅作为背景参考。本项目的 MoonBit 实现、API 设计、测试和文档均为独立完成。

## 静电势计算结果准确性说明

本项目对“准确性”采用分层定义。

首先，点电荷模式的计算公式本身是准确实现的经典库仑势：

\[
\Phi(\mathbf r)=k\sum_i \frac{q_i}{|\mathbf r-\mathbf R_i|}
\]

在输入坐标为 Angstrom、电荷为基本电荷单位 `e` 时，项目使用库仑常数 `14.3996454784255 eV Angstrom / e^2`，输出单位为 `eV/e`。因此，对于给定的原子部分电荷，该模式的数值计算是确定且可验证的。

但是，点电荷模式不是严格量子化学 MESP。它将连续电子云和核电荷共同产生的势场简化为原子中心单极电荷，无法真实描述孤对电子、π 体系、σ-hole、分子内部区域以及近核区域的各向异性静电势。因此，如果用户期望得到严格量子化学意义上的分子表面静电势，不能只使用部分电荷点电荷模型。

其次，严格 MESP 后处理模式采用公认量子化学公式：

\[
V(\mathbf r)=k\left[\sum_A \frac{Z_A}{|\mathbf r-\mathbf R_A|}
-\int \frac{\rho(\mathbf r')}{|\mathbf r-\mathbf r'|}d\mathbf r'\right]
\]

该模式的准确性主要取决于外部输入的电子密度质量、网格分辨率、积分区域大小和单位一致性。只要输入电子密度来自可靠的 Hartree-Fock 或 DFT 计算，并且网格足够细、覆盖范围足够大，本项目的数值积分结果可作为严格 MESP 的后处理近似。项目本身不宣称能从零求解电子结构。

## 静电势可视化准确性验证方案

1. 解析解验证
   - 使用单个点电荷测试 `V=kq/r`。
   - 使用正负等量电荷测试对称点处电势抵消。
   - 使用单核电荷测试核项 `kZ/r`。
   - 使用简单均匀电子密度网格测试电子贡献符号必须为负。

2. 单位与积分验证
   - 验证 Bohr 与 Angstrom 的密度单位换算：
     `rho(a.u.) = rho(e/Angstrom^3) * a0^3`。
   - 验证电子密度网格积分得到的总电子数：
     \[
     N_e=\sum_g \rho_g \Delta V_g
     \]
   - 对网格加密进行收敛性测试，检查 MESP 数值随网格间距减小趋于稳定。

3. 与成熟量子化学工具对比
   - 使用 Gaussian、ORCA、Psi4、CP2K 或 Multiwfn 生成同一分子的电子密度与静电势参考数据。
   - 选取固定测试点、二维切片平面和电子密度等值面采样点，比较本项目输出与参考工具输出。
   - 误差指标包括最大绝对误差、均方根误差、相关系数和正负区域一致性。

4. 分子表面验证
   - 严格 MESP 表面必须使用电子密度等值面，默认 `rho = 0.001 a.u.`。
   - 验证表面顶点是否满足目标电子密度阈值。
   - 分别在表面顶点计算 MESP，并与 Multiwfn 等工具导出的表面静电势进行逐点或插值对比。
   - 范德华表面仅作为交互式近似展示，不用于宣称严格 MESP 表面结果。

5. 颜色映射验证
   - 验证 0 电势映射为白色，负电势映射为红色，正电势映射为蓝色。
   - 对跨分子比较使用固定对称色标，例如 `[-Vmax, +Vmax]`，避免自动缩放造成视觉误判。
   - 检查图例范围与实际用于表面和切片的色标一致。
   - 对同一数据集导出 CSV 数值和 SVG/3D 颜色结果，确认颜色与数值一一对应。

6. 科学合理性案例验证
   - 水分子中氧端附近应呈相对负电势，氢端附近应呈相对正电势。
   - CO2 在线性分子轴向和侧向区域应体现符合分子对称性的静电势分布。
   - 甲醛羰基氧附近应呈负电势，羰基碳附近应体现亲核进攻相关的正电势区域。
   - 对含孤对电子或 σ-hole 特征的体系，应使用电子密度 MESP 模式验证，不能依赖简单点电荷模式。

综上，本项目的点电荷模式适合快速、稳定、可解释的近似可视化；严格 MESP 模式在外部电子密度输入可靠的前提下，符合真实量子化学理论公式。项目将通过解析测试、单位测试、积分收敛测试、成熟工具对比和颜色映射一致性检查，系统验证静电势计算和可视化结果的准确性。
